개인정보처리방침 — 매일매일 수학

시행일: 2026-07-16 · 문의: iepnote@gmail.com

「매일매일 수학」(이하 "앱")은 자폐성장애·지적장애 초등학생을 위한 연산 학습 앱입니다. 앱은 회원가입과 계정이 없으며, 개인을 식별할 수 있는 정보를 수집하지 않는 것을 원칙으로 설계되었습니다.

1. 수집·저장하는 정보

항목내용비고
교실 코드무작위 6자리 문자개인정보 아님(교사가 생성)
학생 표시 이름교사가 입력한 별명·이니셜앱 안내문에서 실명 사용을 권장하지 않음
학습 기록학습 단계, 정답률, 완료 날짜, 문제별 풀이 시간학습 지도 목적
교사 PINSHA-256 해시로만 저장원문 저장 안 함

이메일, 전화번호, 주소, 위치, 사진, 연락처, 생년월일 등 개인 식별 정보는 수집하지 않습니다. 아동에게서 직접 정보를 수집하지 않으며, 모든 입력(교실·학생 등록)은 교사가 수행합니다.

2. 저장 위치와 보안

  • 기본적으로 정보는 사용 기기 안(localStorage)에 저장됩니다.
  • 여러 기기 진도 공유를 위해 온라인 저장소(Supabase, 암호화 통신 HTTPS)를 사용하며, 교실 코드 단위로 데이터가 격리됩니다.
  • 교실 코드를 아는 사람만 해당 교실 데이터에 접근할 수 있으므로, 코드는 학급 구성원 외 공유하지 않도록 안내합니다.

3. 이용 목적

수집한 정보는 학습 진도 저장, 완료 달력 표시, 교사의 학습 지도(대시보드)에만 사용됩니다.

4. 제3자 제공·광고·추적

  • 수집 정보를 제3자에게 제공하지 않습니다.
  • 광고가 없으며, 광고 SDK·분석 도구·추적 쿠키를 사용하지 않습니다.

5. 보관 기간과 삭제

  • 교사는 대시보드에서 학생과 학습 기록을 언제든 삭제할 수 있습니다.
  • 교실 전체 삭제를 원하시면 위 문의 이메일로 교실 코드를 알려 주세요. 지체 없이 삭제합니다.

6. 아동의 개인정보 보호

이 앱은 아동이 직접 개인정보를 입력하는 화면이 없습니다. 학생 등록·이름 표기는 보호자 또는 교사가 하며, 별명·이니셜 사용을 권장하는 안내문이 앱 안에 표시됩니다.

7. 방침의 변경

방침이 바뀌면 이 페이지에 변경 내용과 시행일을 게시합니다.
